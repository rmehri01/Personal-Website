Projects

Big Data Weather Analysis
(Functional Programming in Scala Capstone)

Scala | Apache Spark | Amazon AWS (EMR and S3)

This project is an analysis and visualization of gigabytes of weather data from over 40 years. It applies data processing skills to a relatively data-intensive application using real-world data from the National Center for Environmental Information of the United States. It involves cleaning raw data, extracting useful information from it, and using spatial and linear interpolation techniques to visualize the data. An example of this is finding the average temperature or deviation in temperature of each point on the globe over the year.

Recipe Nest
(nwHacks 2020 Winner of "Voiceflow - Best Voice App")

Google Cloud Dialogflow | Python | Flask | Spoonacular API

A recipe finding chat bot that helps reduce food waste! Users simply enter in all the filters including diet, cuisine, ingredients, etc. and Recipe Nest finds a recipe conforming to the users' requests! We deployed this chat bot on many different platforms including as a web app, on Slack, on call, and have requested approval for google assistant and messenger. We believe that making this application as accessible as possible reflects our goal of making it easy to get started with cooking at home and not wasting food!

UBC Housing Finder

Scala | Play Framework | Jsoup | HTML | CSS

This is a web app that I made to help UBC students find housing around off-campus in the Vancouver area. Visiting numerous different sites to do so is time consuming and quite repetitive. This web app aims to be one place to visit where students can quickly see the specifications of each listing on one page according to their housing preferences. Check it out on the title link!

Sign Language Translator
(Local Hack Day 2019)

Python | Flask | Google Cloud Text-to-Speech | HTML | CSS

A web app that quickly translates either speech or text to ASL. For common phrases, it returns a picture describing the motion, while other words are directly translated using the ASL alphabet. A Google Cloud Speech-to-Text API that utilizes neural networks is able to translate speech with astounding accuracy.
